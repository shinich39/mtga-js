# mtga-js

Make Textarea Garbage Again

## Features

- Commentify: Ctrl+/
- Indentify: Tab, Shift+Tab
- Auto-Complete
- Auto-Pairing: All brackets with ", ', `
- Undo, Redo: Ctrl+Z, Ctrl+Shift+Z
